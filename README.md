# AIAgent
## 🛠 技術スタック (Technical Stack)

本アプリケーションは、モダンなフロントエンドフレームワークと高速なAI・軽量バックエンド、および外部のリアルタイム気象APIを一本のパイプラインで統合した、位置情報連動型のAIスタイリスト（コーディネート提案）システムです。

### 💻 フロントエンド (Frontend)
* **Next.js (v14+)**: App Routerを採用し、高速な画面遷移とコンポーネントベースの堅牢なUIを構築。
    * `use client` を用いたインタラクティブな状態管理（ユーザー位置情報および気象データの即時同期）。
* **TypeScript**: 型安全性を担保し、APIレスポンスやコンポーネント間のデータ受け渡し（Props）におけるバグを徹底排除。
* **Tailwind CSS**: ニュアンスカラー（ブルー・ホワイト・ピンク）の美しいグラデーション、背景のグラスモフィズム（backdrop-blur）、ホバー時の滑らかなアニメーションなどを施した、洗練されたモダンなUXデザイン。
* **React Markdown**: バックエンド（LLM）から返却されるMarkdown形式のテキストクレンジング、およびマークアップ表示の最適化。

### ⚙️ バックエンド (Backend)
* **FastAPI**: Python製の超高速・軽量な非同期Webフレームワーク。フロントエンドからのリクエスト（ユーザープロンプト＋現在地気象データ）を高速に処理。
* **Uvicorn**: ASGIサーバーとして動作し、ローカル環境（`http://127.0.0.1:8000`）での安定したホットリロードと高パフォーマンスな通信を実現。

### 🤖 生成AI / RAG (AI & Retrieval-Augmented Generation)
* **RAG (Retrieval-Augmented Generation)**: 
    * 外部気象APIから取得した「都市名・リアルタイム気温・最低/最高気温・気候ステータス」の生データを、AIへのプロンプトに動的に注入。
    * 固定値やハルシネーション（嘘の回答）を排除し、ユーザーの「今、その場所」の環境に100%パーソナライズされた、コンテキスト精度の高いコーディネート提案ロジックを実現。

### 📡 外部API & ブラウザインフラ
* **OpenWeatherMap API**: 緯度・経度ベースで世界中のリアルタイム気象データを取得する外部インフラ。
* **HTML5 Geolocation API**: ブラウザ標準のGPSインフラ機能。ユーザーの許可を得て端末の現在地（緯度・経度）を安全かつミリ秒単位で瞬時にハック。
